## 1. Начало работы

### 1.1 Регистрация

1. Откройте сайт в браузере.
2. Нажмите **«Зарегистрироваться»** на странице входа.
3. Заполните поля:
   - **Имя** — как к вам обращаться
   - **Email** — на него придёт письмо с подтверждением
   - **Пароль** — минимум 8 символов
4. Нажмите **«Зарегистрироваться»**.
5. Откройте письмо, которое пришло на ваш email, и нажмите **«Подтвердить email»**.
6. После подтверждения вы можете войти в систему.

> Если письмо не пришло — проверьте папку «Спам». Можно запросить повторную отправку на странице входа.

### 1.2 Вход в систему

1. Введите email и пароль.
2. Нажмите **«Войти»**.
3. Если включён PIN-код — введите его (настраивается в приложении).

### 1.3 Восстановление пароля

1. На странице входа нажмите **«Забыли пароль?»**.
2. Введите email, к которому привязан аккаунт.
3. Откройте письмо — вы получите код для восстановления.
4. Введите код на странице сброса пароля.
5. Придумайте новый пароль и подтвердите его.

## 2. Главный экран — Файлы

После входа вы попадаете на страницу **«Файлы»** — это ваш основной рабочий экран.

### 2.1 Интерфейс

- **Боковое меню** — навигация по разделам
- **Верхняя панель** — поиск, уведомления, настройки
- **Центральная область** — список ваших файлов
- **Счётчик хранилища** — сколько места занято / доступно

### 2.2 Фильтры

Вверху списка файлов доступны вкладки:

| Вкладка | Что показывает |
|---------|---------------|
| **Мои файлы** | Файлы, которые вы загрузили |
| **Полученные** | Файлы, которыми с вами поделились |
| **Избранное** | Файлы, отмеченные звёздочкой |
| **Все** | Все файлы, к которым у вас есть доступ |

### 2.3 Поиск

В строке поиска можно искать файлы по названию. Доступна фильтрация:
- по папке
- по тегу
- по типу файла

## 3. Загрузка и скачивание файлов

### 3.1 Загрузка файла

1. Нажмите кнопку **«Загрузить»** на странице «Файлы».
2. Выберите один или несколько файлов на вашем устройстве.
3. Дождитесь окончания загрузки — появится зелёная отметка.
4. Файл появится в общем списке.

> Поддерживаются файлы любых типов. Размер файла ограничен вашим тарифным планом.

### 3.2 Добавление ссылки как файла

Вместо загрузки файла можно сохранить ссылку на веб-страницу:

1. Нажмите **«Добавить ссылку»**.
2. Вставьте URL веб-страницы.
3. Нажмите **«Предпросмотр»** — система покажет заголовок и описание страницы.
4. Нажмите **«Сохранить»** — ссылка появится в списке файлов.

### 3.3 Скачивание файла

1. Нажмите на файл, чтобы открыть его карточку.
2. Нажмите кнопку **«Скачать»**.
3. Браузер начнёт скачивание файла.

> Скачивать можно только те файлы, к которым у вас есть доступ.

## 4. Работа с файлами

### 4.1 Карточка файла

Нажмите на любой файл, чтобы открыть его детальную карточку. Здесь доступны:

- **Просмотр** — если файл изображение или видео
- **Информация** — размер, тип, дата загрузки, владелец
- **Описание** — можно добавить заметку к файлу
- **Теги** — назначить метки для удобного поиска
- **Активность** — история действий с файлом

### 4.2 Избранное

Нажмите на звёздочку у файла, чтобы добавить его в избранное. Это поможет быстро находить важные файлы через фильтр «Избранное».

### 4.3 Закрепление (Pin)

Закреплённые файлы помечаются особым значком. Файлы других пользователей, которые вы сохранили через ссылку, также отмечаются как закреплённые.

### 4.4 Перемещение в папку

1. В карточке файла нажмите **«Переместить»**.
2. Выберите папку из дерева.
3. Файл переместится в указанную папку.

### 4.5 Удаление файла

1. В карточке файла нажмите **«Удалить»**.
2. Подтвердите удаление.

> Если вы владелец — файл помечается как удалённый. Если файл был вам доступен по шерингу — он просто исчезнет из вашего списка.

## 5. Организация файлов

### 5.1 Папки

На странице **«Папки»** отображается дерево папок.

- **Создать папку** — укажите название и, при необходимости, родительскую папку
- **Переименовать** — нажмите на папку и выберите «Переименовать»
- **Удалить** — можно удалить только пустую папку; если в ней есть файлы, их нужно переместить
- **Перемещать файлы** — перетащите файл в нужную папку

### 5.2 Теги

На странице **«Теги»** вы управляете метками для файлов.

- **Создать тег** — задайте название (например, «Работа», «Личное», «Фото»)
- **Назначить файлу** — откройте карточку файла и выберите теги
- **Фильтр по тегу** — нажмите на тег, чтобы увидеть все файлы с этой меткой

### 5.3 Поиск

Используйте строку поиска в верхней панели. Можно комбинировать:
- поиск по названию
- фильтр по папке
- фильтр по тегу
- фильтр по типу файла (документы, изображения, видео и т.д.)

## 6. Шеринг файлов

### 6.1 Поделиться с контактом

1. Откройте карточку файла.
2. Нажмите **«Поделиться»**.
3. Выберите контакт из адресной книги.
4. Файл появится в разделе «Полученные» у выбранного контакта.
5. Контакт получит push-уведомление (если включено).

> Если контакт ещё не зарегистрирован в системе — файл будет доставлен автоматически после его регистрации и принятия приглашения.

### 6.2 Отозвать доступ

1. В карточке файла нажмите **«Доступ»**.
2. Найдите контакт в списке и нажмите **«Отозвать»**.

### 6.3 Публичная ссылка

1. Откройте карточку файла.
2. Нажмите **«Создать ссылку»**.
3. Настройте параметры:
   - **Срок действия** — через сколько часов ссылка перестанет работать (от 1 до 720 часов)
   - **Разрешить сохранение** — если включено, другие пользователи смогут сохранить файл к себе
4. Нажмите **«Создать»**.
5. Скопируйте получившуюся ссылку и отправьте кому угодно.

### 6.4 Просмотр по ссылке (неавторизованные пользователи)

По ссылке любой человек может:
- увидеть название и размер файла
- скачать файл
- сохранить файл в свой аккаунт (если разрешено)

### 6.5 Управление ссылками

В карточке файла на вкладке **«Ссылки»** вы можете:
- посмотреть все созданные ссылки
- отключить любую ссылку досрочно

## 7. Общие папки

Общие папки позволяют работать с файлами нескольким пользователям совместно.

### 7.1 Создание общей папки

1. Перейдите в раздел **«Общие папки»**.
2. Нажмите **«Создать папку»**.
3. Задайте название и нажмите **«Создать»**.

### 7.2 Добавление участников

1. Откройте общую папку.
2. Нажмите **«Доступ»** → **«Добавить»**.
3. Выберите контакт и уровень доступа:
   - **Просмотр** — может только смотреть и скачивать файлы
   - **Редактирование** — может загружать, удалять и изменять файлы

### 7.3 Загрузка в общую папку

1. Откройте общую папку.
2. Нажмите **«Загрузить»**.
3. Выберите файлы. Они будут доступны всем участникам папки.

### 7.4 Добавление в «Мои файлы»

Если вы участник общей папки и хотите, чтобы файл был всегда под рукой:
1. Нажмите на файл.
2. Выберите **«Добавить в мои файлы»**.
3. Файл появится в вашем личном списке.

### 7.5 Ссылка на общую папку

Как и для обычных файлов, можно создать публичную ссылку на всю общую папку.

## 8. Контакты

### 8.1 Добавление контакта

1. Перейдите в раздел **«Контакты»**.
2. Нажмите **«Добавить контакт»**.
3. Введите имя и email (или номер телефона).
4. Если пользователь уже зарегистрирован — ему придёт запрос на добавление в контакты.
5. После подтверждения контакт появится в вашем списке.

### 8.2 Импорт контактов

Можно загрузить список контактов из адресной книги телефона или другого сервиса (до 500 записей за раз).

### 8.3 Запросы в контакты

Если кто-то добавил вас в контакты:
1. В разделе «Контакты» откройте вкладку **«Запросы»**.
2. Нажмите **«Принять»** или **«Отклонить»**.

### 8.4 Приглашения

Если ваш контакт ещё не зарегистрирован:
1. Нажмите **«Пригласить»** рядом с его именем.
2. Система отправит приглашение на email.
3. Когда он зарегистрируется и примет приглашение — контакт автоматически активируется, а все отложенные файлы будут доставлены.

### 8.5 История обмена

В карточке контакта можно посмотреть историю — какие файлы вы ему отправляли и получали от него.

## 9. Ссылки (URL-файлы)

### 9.1 Сохранение веб-ссылки

Вместо загрузки файла можно сохранить ссылку на любой сайт:

1. На странице «Файлы» нажмите **«Добавить ссылку»**.
2. Вставьте URL.
3. Система покажет предпросмотр — заголовок, описание и картинку с сайта.
4. Нажмите **«Сохранить»**.
5. Ссылка появится в списке как обычный файл.

### 9.2 Открытие ссылки

Нажмите на URL-файл → **«Открыть»** → сайт откроется в новой вкладке.

## 10. Лента активности

В разделе **«Активность»** отображается хронология всех действий с вашими файлами:

- загрузка файлов
- скачивание
- отправка контактам
- создание / отключение ссылок
- закрепление / открепление
- добавление в избранное
- удаление
- получение файлов от других

Каждое событие содержит дату, название файла и тип действия.

|----------|------|--------|------|

### 11.2 Просмотр использования

В разделе **«Тарифы»** отображаются:
- сколько места занято / доступно (графическая шкала)

### 11.3 Смена тарифа

Нажмите **«Сменить тариф»** и выберите новый план. Запрос будет направлен администратору.

## 12. Поддержка и предложения

### 12.1 Создание тикета поддержки

1. Перейдите в раздел **«Поддержка»**.
2. Нажмите **«Создать тикет»**.
3. Опишите проблему.
4. При необходимости прикрепите скриншоты (изображения до 10 МБ).
5. Нажмите **«Отправить»**.

### 12.2 Общение в тикете

- Вы можете добавлять новые сообщения в открытый тикет.
- Когда администратор ответит, вы получите уведомление.
- Если проблема решена — нажмите **«Подтвердить»**.
- Тикет закроется автоматически.

### 12.3 Предложения

В разделе «Поддержка» есть вкладка **«Предложения»**:
1. Напишите идею по улучшению сервиса.
2. При необходимости прикрепите изображения.
3. Администратор может принять предложение или оставить комментарий.

## 13. Настройки и безопасность

### 13.1 Смена пароля

1. Перейдите в **«Настройки»** → **«Безопасность»**.
2. Введите текущий пароль.
3. Введите новый пароль дважды.
4. Нажмите **«Сохранить»**.

### 13.2 Смена email

1. В настройках безопасности нажмите **«Сменить email»**.
2. Введите новый email.
3. Подтвердите новый адрес через письмо.

### 13.3 Управление сессиями

В разделе «Безопасность» отображаются все активные сессии:
- **Устройство** — название браузера / приложения
- **IP-адрес**
- **Последняя активность**

Вы можете завершить отдельную сессию или все сразу (например, если потеряли телефон).

### 13.4 PIN-код

Для дополнительной защиты можно установить PIN-код:
1. После входа в систему вам будет предложено установить PIN.
2. PIN-код хранится локально на устройстве, не отправляется на сервер.
3. При последующих входах достаточно ввести PIN.

### 13.5 Уведомления

В настройках можно включить / отключить:
- уведомления о новых файлах
- уведомления о добавлении в контакты
- push-уведомления (всплывающие в браузере)

## 14. Мобильное приложение (PWA)

DeliFile — это Progressive Web Application (PWA). Это значит, что оно работает как обычное мобильное приложение, но не требует установки из магазина.

### 14.1 Установка на телефон

**Android (Chrome):**
1. Откройте сайт в Chrome.
2. Нажмите на меню (три точки) → **«Установить приложение»**.
3. Иконка появится на рабочем столе.

**iOS (Safari):**
1. Откройте сайт в Safari.
2. Нажмите на кнопку «Поделиться» (квадрат со стрелкой).
3. Выберите **«На экран «Домой»**.
4. Иконка появится на рабочем столе.

### 14.2 Поделиться файлом из другого приложения

1. В любом приложении нажмите **«Поделиться»**.
2. Выберите DeliFile из списка.
3. Файл автоматически загрузится в ваш аккаунт.

### 14.3 Push-уведомления

При включённых уведомлениях вы будете получать оповещения:
- когда кто-то поделился с вами файлом
- когда вам ответили в тикете поддержки

## 15. Часто задаваемые вопросы

**Q: Какой максимальный размер файла можно загрузить?**
Зависит от тарифа: Free — 50 МБ, Silver — 100 МБ, Gold — 150 МБ.

**Q: Как долго хранятся файлы?**
Файлы хранятся, пока вы их не удалите, или пока не истечёт срок их действия (если он был установлен).

**Q: Может ли кто-то увидеть мои файлы без моего разрешения?**
Нет. S3-хранилище полностью приватное. Доступ к файлу есть только у вас и тех, кому вы явно дали доступ.

**Q: Как удалить файл окончательно?**
После удаления файл помечается как удалённый. Система полностью очищает его в течение часа.

**Q: Что происходит, когда заканчивается место?**
Вы не сможете загружать новые файлы, пока не освободите место или не перейдёте на тариф с большим объёмом.

**Q: Как отозвать доступ к файлу?**
Откройте карточку файла → вкладка «Доступ» → нажмите «Отозвать» рядом с контактом.

**Q: Можно ли восстановить удалённый файл?**
Нет, удаление необратимо. Будьте внимательны.

**Q: Как выйти из аккаунта на всех устройствах?**
Настройки → Безопасность → «Завершить все сессии».

**Q: Безопасно ли сохранять ссылки на файлы?**
Да. Ссылки защищены уникальным токеном. Вы можете отключить любую ссылку в любой момент.
