## 8. Маркер «Смотрел» + личный рейтинг + фильтры в папке Фильмы

### Backend
- Новый endpoint `PATCH /api/v1/files/{id}/movie-meta`
  - Поля: `watched: bool|null`, `personal_rating: float 0–10|null`
  - Обновляет только эти ключи в `custom_metadata`, не затрагивая данные Кинопоиска
- При Kinopoisk-обновлении — сохранять `watched`/`personal_rating` из старого `custom_metadata`
- Маршрут в `routes/api.php`

### Web (`movie-view.component.ts`)
- На карточке: иконка 👁 (watched toggle) + личный рейтинг `0–10`
- Просмотренные — приглушённая карточка
- Над списком фильмов в `folders-tree.component.html`:
  - Фильтр: «Все» / «Смотрел» / «Не смотрел»
  - Фильтр/сортировка по личному рейтингу
  - Фильтр/сортировка по рейтингу Кинопоиска
- Фильтрация клиентская

### Mobile (`shared-folders/[id].tsx` + `MovieCard.tsx`)
- Кнопка 👁 и поле рейтинга на карточке
- Горизонтальные чипы-фильтры над `FlatList`
- Сохранение через `PATCH /api/v1/files/{id}/movie-meta`
